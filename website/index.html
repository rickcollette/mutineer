MUTINEER BBS

A classic telnet BBS — written in C  ▮

Features Documentation Install GitHub

Sail the digital seas.

Mutineer is a full-featured telnet BBS server built from scratch in C11, styled after the classic Renegade BBS with a green pirate theme. SQLite persistence, DOS doors, PLANK networking, and Buccaneer, an interpreted language for addons, games, doors, and extensions.

Features

Multi-Node Telnet

Thread-per-connection, up to 256 nodes. WFC console, whisper, wall broadcasts.

Message System

Threaded boards, private email, QWK offline mail, FidoNet netmail, FSEditor.

File Areas

Upload/download, credits/ratio enforcement, batch ops, archive test/extract.

DOS Door Runner

DOSBox with serial nullmodem socket passthrough. Aladdin Adventure included.

ACS System

Full expression parser: levels, flags, ratios, time, credits, conferences.

PLANK Networking

Store-and-forward federation with Ed25519 signatures and CBOR objects.

Buccaneer

Interpreted language for interactive doors, addons, and games. Pascal-like syntax.

Plugins

Loadable .so modules via dlopen with capability-controlled host API.

Quick Start

# Docker (Linux, macOS, Windows) git clone https://github.com/rickcollette/mutineer.git cd mutineer && docker compose up -d telnet localhost 2929 # Or build from source (Linux / WSL) cmake -S . -B build -DCMAKE_BUILD_TYPE=Release cmake --build build build/mutineer-initbbs -c conf/mutineer.conf -y build/mutineer -c conf/mutineer.conf
